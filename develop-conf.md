#### 安装oh-my-zsh
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
windows用户需要修改 ~/.zshrc 找到并设置ZSH_THEME="ys"否则有个光标的小bug

```
vim ~/.zshrc
# 修改找到并设置ZSH_THEME 后保存退出
source ~/.zshrc
```
